# 项目简介
一个简单的备忘录，使用了go + Hertz + gorm + MySQL实现

# 项目功能
- **user模块实现基本的用户注册登录 ( 用token实现 )**
  
- **task模块实现增、改、查、删四种基本功能**

**增**
- 添加一条新的待办事项

**改**
- 将 一条/所有 代办事项设置为已完成
- 将 一条/所有 已完成事项设置为待办

**查**

- 查看所有 已完成/未完成/所有 事项。 
- 输入**关键词**查询事项。
- 可进行分页

**删**

- 删除 一条/所有已经完成/所有待办/所有 事项
